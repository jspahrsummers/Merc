# Merc
A massively multiplayer space sim inspired by the [Escape Velocity](https://en.wikipedia.org/wiki/Escape_Velocity_(video_game)) series of video games. For more modern inspiration (still single player), see [Endless Sky](http://endless-sky.github.io/)--but note that its license does not permit direct source code influence or reuse of assets.

The [Merc wiki](https://github.com/jspahrsummers/Merc/wiki) contains semi-organized thoughts about how the game should come together, and ways in which it might differ from the aforementioned games.
